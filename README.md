# Privacy Mode for Trade Republic

A browser extension to hide your account balances on the Trade Republic web app using a discrete "dots" mask.

## Installation (Manual Load)
1. Download this repository as a ZIP file and extract it.
2. Open Brave or Chrome and go to `brave://extensions` or `chrome://extensions`.
3. Enable **Developer Mode** (top right toggle).
4. Click **Load unpacked** and select the folder containing these files.
5. Open Trade Republic Web – your balances are now hidden by default!

## Features
- Replaces €/$ amounts with `••••`.
- Percentages remain visible.
- Toggle button in the bottom right corner.
- Settings are saved automatically.
